Tutorials
=========

These tutorials are a quick start on how to use ROSS.
Part 1 introduces to how to create and assign materials and elements, how to
build a rotor model and manage units.
Part 2 introduces to the analyzes presented in ROSS, how to run each one
and the possibilities of post-processing results.
Part 3 introduces to the ROSS' stochastic analyis package. A quick start on
how to run statistical rotordynamic analysis.
